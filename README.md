# ThemeHang v2 — Spotlight

A free, mobile-first daily word puzzle game playable in any browser.

## How it works

Five hidden words share a theme — Movies, Books, or Capital Cities. 
Type a real English word to guess. Two things can happen:

- ✅ **Exact match** — the word is solved and turns green
- ✦ **Right length, wrong word** — any letters from your guess that appear 
  in the remaining hidden words get spotlighted in gold, permanently

You have 12 attempts. A cute hangman builds up with each one.

## Built with

- Vanilla HTML, CSS and JavaScript — no frameworks, no dependencies
- [Free Dictionary API](https://dictionaryapi.dev) for word validation
- Hosted free on GitHub Pages

## Features

- 3 categories: 🎬 Movies · 📚 Books · 🌍 Capital Cities
- 4 puzzles per category (16 total)
- Dictionary-validated guesses — no random strings accepted
- Hint coin system and Pro subscription ready for monetisation
- Fully mobile responsive
